### 3.项目说明：
#### 本项目使用了React的脚手架来快速搭建一个React项目，项目的UI界面采用了Ant Design，让React小白能够快速掌握React的开发流程

#### 项目中使用了当前主流的Redux进行数据存储，只用了一个Redux的例子，让未接触过项目的小白快速了解如何使用Redux以及react-redux将Redux在React项目中进行使用
